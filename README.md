WUCUPID is a sophisticated matchmaking platform designed specifically for WUSTL students. The application automates the process of finding compatible matches based on personality traits, interests, and preferences. Using a cyclical matching system, WUCUPID collects survey responses during designated periods, processes the data using AI-powered compatibility analysis, and presents users with their mutual matches.

Key Features:
****
Cyclic Matching System: Scheduled phases (survey collection, processing, match viewing)


AI-Powered Matching: Uses GPT-4o to evaluate user compatibility


User Authentication: Secure login and account management


Comprehensive Survey: Collects personality traits and preferences


Match Descriptions: AI-generated profiles for each match
